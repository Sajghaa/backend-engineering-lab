# Auth System – FastAPI

A production-ready authentication API built with FastAPI, SQLAlchemy, JWT and bcrypt.

## Features
- User registration and login
- JWT access tokens
- Password hashing
- Role-based access control (user / admin)
- Clean architecture (routes, services, models)
- Unit tests
